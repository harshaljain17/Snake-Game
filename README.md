A sleek, logic-driven recreation of the classic Snake game built with Vanilla JavaScript and CSS Grid. This project focuses on clean state management and efficient DOM manipulation, utilizing a coordinate-based grid system to handle movement and collision detection. Key features include a buffered input system to prevent illegal 180-degree turns, dynamic food generation that avoids the snake's body, and a fully responsive game-over state. Whether you are looking to kill time or study game loop logic, this project demonstrates the power of modern web standards without the need for external frameworks.

Key Features
Coordinate-Based Grid: Dynamically calculates rows and columns based on viewport size for a perfect fit on any screen.

Collision Logic: Robust detection for both wall impacts and self-collision to ensure a challenging experience.

Input Buffering: Prevents "instant death" by ensuring the snake cannot turn directly back on itself during high-speed play.

Responsive UI: Features a modal-based start and game-over system using fixed and absolute positioning for a polished user experience.
